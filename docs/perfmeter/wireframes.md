# Wireframes

## Dashboard
```
+-------------------------------------------------------------+
| Performance Meter             [Total Time] [WPM] [Score]    |
+----------------------+--------------------+-----------------+
| Typing & Input       | Gemini Evaluation  |                 |
|  Words  Backspaces   |  Grade  Notes      |                 |
|  Keys   Mouse Dist   |  Rationale         |                 |
+-------------------------------------------------------------+
| Time by App (Bar Chart)                                    |
+-------------------------------------------------------------+
| Stress Analysis (N days): Level | Score | Conf | Signals    |
+-------------------------------------------------------------+
```

## Ctrl+C Flow
- Terminal prints JSON summary.
- Browser auto-opens dashboard with “waiting for AI evaluation”.
- Once Gemini completes, score/grade/notes appear.
